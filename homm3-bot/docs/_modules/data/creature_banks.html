        """
        Init function of the Creature Bank Class.

        :param value: Value assigned to Creature Bank
        :param name: Name of the creature bank
        :param visited: Boolean. Visited -> True else False
        :param pair:

        self.value = value
        self.name = name

        self.guard_reward = pair
